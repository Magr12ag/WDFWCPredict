# WDF World Cup Predictions Dashboard

Static HTML dashboard for GitHub Pages.

## GitHub Pages upload

Upload these files at the repository root:

- `index.html`
- `.nojekyll`
- `README.md`

Then enable GitHub Pages from **Settings > Pages > Deploy from a branch > main > /(root)**.

## Notes

- The dashboard is self-contained and has no external dependencies.
- Scores are stored locally in each visitor's browser via `localStorage`.
- The refresh button is prepared for a future results API, but no API URL is configured yet.
