# WDF World Cup Prediction Dashboard

Simple GitHub Pages version.

## Upload these files to GitHub

Upload all files in this folder to the root of your repository:

- index.html
- .nojekyll
- results.json
- README.md

## Important

Do not upload the ZIP file itself.
Extract the ZIP and upload the files inside it.

## GitHub Pages settings

Go to:

Settings → Pages

Use:

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

## Scoring

This simple version uses manual scoring:

1. Open the dashboard.
2. Go to the match roster.
3. Type the score into a match.
4. The leaderboard updates automatically.

Scores are saved in the browser's local storage.
