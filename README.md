# WDF World Cup Prediction Dashboard

Simple GitHub Pages version.

## Upload these files to GitHub

Upload all files in this folder to the root of your repository:

- index.html
- .nojekyll
- results.json
- README.md

## GitHub Pages settings

Settings → Pages

Use:

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

## Scoring

This version uses manual score entry:

1. Open the dashboard.
2. Go to the match roster.
3. Type scores into matches.
4. The leaderboard updates automatically.
5. Click "Refresh scores" to re-render the dashboard and show "Refresh successful".

Scores are saved in each browser's local storage.
