# WDF World Cup 2026 Prediction Dashboard

GitHub Pages-ready dashboard.

## Public API
This version uses the CORS-enabled public endpoint:
https://wheniskickoff.com/data/v1/matches.json

Important: this public endpoint is safest for GitHub Pages/browser access. If no live score fields are available yet, manual score entry remains active.

## Deploy
Upload these files to the root of your GitHub repo:
- index.html
- .nojekyll
- README.md

Then enable GitHub Pages:
Settings → Pages → Deploy from branch → main → / (root)

After uploading, open your site with `?v=3` added to the URL to bypass browser cache.
